\section*{Contenuti formativi previsti}
% Personalizzare indicando le tecnologie e gli ambiti di interesse dello stage
Durante questo progetto di stage lo studente avrà occasione di approfondire le proprie conoscenze nell'ambito dello sviluppo di applicazioni web full-stack, con particolare riferimento alla progettazione e realizzazione di interfacce utente tramite il framework Angular e allo sviluppo di servizi backend basati sul framework Spring. Il progetto consentirà inoltre di consolidare competenze relative alla progettazione di API secondo il paradigma REST, nonché alla gestione della comunicazione client-server. In caso di implementazione della funzionalità opzionale di chat in tempo reale, lo studente avrà l'opportunità di esplorare protocolli di comunicazione alternativi come WebSocket.
